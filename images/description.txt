# Шрифты
Headline font - EB Garamond Regular 140px ;
Description text - Montserrat Light 16px (в макете будет встречаться Gilroy Light - заменить на Montserrat)

# header
По клику на сэндвич, открывается меню ( стилизация и внутренние ссылки на Ваше усмотрение ), сендвич меняется на крестик, по клику на который меню закрывается. Так же, при клике на любую область, кроме меню, оно(меню) должно закрываться.

Логотип строго по центру экрана.
По клику на иконку выбора языков, выпадает список из доступных языков (UA,EN,IT). При клике на один из них скрывается список языков и выбранный появляется в данной иконке.

При клике на иконку login(иконка с человечком =) ) появляется модальное окно расположенное по центру экрана, сделанное в стилистике сайта.
В модальном окне:
-заголовок ("Регистрация")
-поля ввода (Имя, Email, Телефон). Поле телефон - с маской ( код Украины).
    * все поля обязательны к заполнению.
- кнопка "Отправить" (при клике на кнопку "Отправить", нужно проверять поля формы на заполнение (валидация)).
-крестик(при клике закрывает модальное окно. Так же, при клике на любую область, кроме модального окна, оно(окно) должно закрываться.)


#footer
Белая полоска на макете - это Footer. Footer, независимо от контента, должен быть прилипшим к низу экрана.

# adaptive
Ваша работа должна быть адаптивной( минимальное разрешение - 320px)

# Будет плюсом:
реализовать sticky header, при скролле страницы.


P.S.
 НЕ использовать свойство float.
 Для меню, выпадающего списка языков и модального окна НЕ использовать никакие плагины, решения на Bootstrap и т.д.!
